# v0.1.10

## 本版亮点

- iOS 现在按 sideload 流程产出 IPA，可以直接给 AltStore / SideStore / 爱思助手这类工具自签安装。
- 发布页继续保留 Android、iOS、Windows 三端下载入口，方便按设备选择。
- Release 附件和下载说明统一了口径，iOS 不再伪装成 App Store/TestFlight 包。

## 体验优化

- iOS 构建改成先生成 `.app`，再打包成 `.ipa`，更贴近开源项目常见的 sideload 方式。
- workflow、README 和 release notes 的描述保持一致，减少下载时的歧义。

## 修复

- 修复 iOS 产物路径写死为签名 IPA 的问题。
- 修复文档里对 iOS 安装方式的表述不一致问题。

## 发布说明

- 本版本由 GitHub Actions 自动构建 Android、iOS 和 Windows 安装包。
- iOS 包会以 sideload IPA 形式出现在 Release 附件中，适合 AltStore / SideStore / 爱思助手自签安装。
