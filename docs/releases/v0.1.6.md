# v0.1.6

## Improved

- Added a public-facing README with clear download buttons for Android and Windows.
- Documented the mobile-first forum experience for non-developer users.

## Fixed

- Parsed real desktop sub-board structures such as C103 child boards.
- Made board child entries more compact and scroll with the thread list instead of behaving like a fixed header.

## Release

- Android APK/AAB and Windows x64 packages are built by the release workflow when pushing tag `v0.1.6`.
- Future releases include split Android APKs for `arm64-v8a`, `armeabi-v7a`, and `x86_64`, plus a universal APK and SHA256 checksums.
