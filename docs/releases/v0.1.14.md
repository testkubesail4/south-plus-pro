# v0.1.14

## 本版亮点

- 一键签到现在会完整走完“领取任务 -> 领取奖励”的整个论坛任务流程，日常和周常不再停在“进行中”。
- 如果任务已经被领取但奖励还没拿，一键签到也会继续识别“进行中”任务并补领奖励。

## 体验优化

- 首页签到状态判断更准确，奖励未领取时不会误显示“已签到”。
- 任务页面里的单个任务操作和一键签到逻辑保持一致，进行中任务会直接走领奖励动作。
- 增加了可开关的论坛任务链路调试日志，后续排查签到异常会更直接。

## 修复

- 修复一键签到只领取任务、不领取奖励，导致任务无法最终完成的问题。
- 修复奖励请求仍按领取任务接口发送，导致进行中任务奖励被漏领的问题。
- 修复“进行中但可领奖”的任务仍被错误判定为已处理、已签到的问题。

## 发布说明

- 本版本继续由 GitHub Actions 自动构建 Android、iOS 和 Windows 安装包。
- Release 页面会继续提供 Android 多架构 APK、AAB、iOS sideload IPA、Windows x64 压缩包和 SHA256 校验文件。
