# v0.1.8

## 本版亮点

- 新增更新检查。以后有新版本时，可以更容易发现并前往下载，不用反复手动翻 Release 页面。
- 应用正式换成“南+”品牌展示，图标和窗口标题都更贴近实际使用场景。
- iOS 构建链路已纳入发布流程，后续可以在 GitHub Release 里同步拿到 IPA。
- 版块列表更好看也更实用。主题预览会根据内容自适应显示，图片、标题和信息排布更清爽。
- 帖子图片加载更顺了。减少大图解码带来的卡顿，刷图和翻帖时更稳。
- 小图显示恢复正常，不会因为优化大图加载而误伤表情、小头像或小尺寸配图。

## 体验优化

- 去掉帖子预览里多余的嵌套边框，列表观感更轻、更接近原生信息流。
- 优化 `thread_new` 版块列表解析，首页和版块页能拿到更准确的主题数据。
- 空父版块现在会给出更清楚的提示，遇到只有子版块的区域时不再显得像加载失败。
- 同人等特殊子版块结构解析更稳定，进入多层版块时少踩空。

## 修复

- 修复部分首页版块结构边界情况，减少版块漏显示、子版块识别不完整的问题。
- 修复空父版块带子版块时的展示和跳转体验。
- 改进图片尺寸控制，降低列表和帖子详情页的渲染压力。

## 发布说明

- 本版本由 GitHub Actions 自动构建 Android、iOS 和 Windows 安装包。
- iOS 包会以 IPA 形式出现在 Release 附件中，适合 App Store Connect 或 TestFlight 流程。
