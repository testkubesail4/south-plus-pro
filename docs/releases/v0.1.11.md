# v0.1.11

## 本版亮点

- 统一了论坛板块与主题链接的解析逻辑，减少不同历史 URL 格式带来的兼容问题。
- 修复从首页进入 `汉化本发布`、`同人音声` 等热门版块时，提示“没有解析到帖子列表”的问题。
- 线程详情页现在能更稳地保留原始分页和锚点信息，避免已定位到某页的链接被错误重写回第一页。

## 体验优化

- 板块链接现在统一支持 `fid-*`、`fid=*`、`fNN`、`pNN` 以及对应的 simple 分页形式。
- simple 模式翻页会保留原始板块模式，不再把图墙类 `pNN` 链接错误改写成普通列表 `fNN`。
- 面包屑里的板块识别也复用了同一套链接语义，详情页返回上下文更稳定。

## 修复

- 修复首页热门版块中的 `汉化本发布`、`同人音声` 无法正常打开的问题。
- 修复部分 simple 板块翻页链接生成错误的问题。
- 修复 `read.php?tid=2876434&page=e#a`、`read.php?tid-2876434-fpage-0-toread--page-7.html` 这类主题链接在详情页解析时丢失原始定位信息的问题。

## 发布说明

- 本版本由 GitHub Actions 自动构建 Android、iOS 和 Windows 安装包。
- Release 页面会继续提供 Android 多架构 APK、AAB、iOS sideload IPA、Windows x64 压缩包和 SHA256 校验文件。
