# v0.1.7

## 本版亮点

- 黑暗模式来了。夜间刷帖更舒服，登录、首页、帖子、搜索、收藏、历史、回复和账号页都做了适配。
- 主题可以自己选。在账号页里切换“跟随系统 / 浅色 / 深色”，不用再被系统设置绑死。
- 论坛任务更省心。首页可以快速领取任务奖励，任务页也支持一键处理。
- 签到类任务会自动申请再领取，少点几次页面，少等几次刷新。
- 任务状态会缓存，刚领过的奖励能更快显示出来，不用每次都重新加载。

## 体验优化

- 任务已经开始、冷却中或需要补一步操作时，应用会尽量自动处理并给出更清楚的状态。
- Android 下载包改为按设备架构提供，主流手机可以下载更小的 ARM64 包；不确定架构时选 Universal 包即可。
- 发布页会显示本版变更、下载说明和校验文件，找版本和安装包更直观。

## 下载提示

- 大多数 Android 手机推荐下载 `south_plus_rewrite-android-arm64-v8a.apk`。
- 不确定设备架构时下载 `south_plus_rewrite-android-universal.apk`。
- Windows 用户下载 `south_plus_rewrite-windows-x64.zip` 后解压运行。
