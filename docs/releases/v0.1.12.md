# v0.1.12

## 本版亮点

- 修复帖子正文里的非方形图片被压扁的问题，正文图和放大预览都会保持原始宽高比。
- 图片手动加载模式下，用户点开帖子图片进入大图预览后会直接加载，不再出现二次点击加载的断层体验。

## 体验优化

- 帖子内联图片的大图预览现在会把“主动点开查看”视为明确加载意图，浏览动作更顺滑。
- 板块列表页同步吸收一次微小的布局整理，避免无意义的代码噪音继续堆积。

## 修复

- 修复帖子正文图片因固定解码宽高而被拉伸压扁的问题。
- 修复手动加载模式下，图片预览弹窗仍显示“点击加载图片”的问题。

## 发布说明

- 本版本由 GitHub Actions 自动构建 Android、iOS 和 Windows 安装包。
- Release 页面会继续提供 Android 多架构 APK、AAB、iOS sideload IPA、Windows x64 压缩包和 SHA256 校验文件。
