# v0.1.9

## 本版亮点

- iOS 构建和发布链路正式打通，Release 里会同时产出 Android、iOS 和 Windows 安装包。
- 发布页补齐了 iPhone / iPad 的 IPA 下载入口，后续测试分发更方便。
- 版本发布说明和下载列表同步更新，能更清楚地看到每个平台对应的产物。

## 体验优化

- 发布流程更统一，Android、iOS 和 Windows 都走同一套 GitHub Release 归档逻辑。
- iOS 现在使用签名和导出配置，减少手工构建时的环境差异。

## 修复

- 修复发布文档里缺少 iOS 下载说明的问题。
- 修复 release 模板和 README 对可发布平台表述不一致的问题。

## 发布说明

- 本版本由 GitHub Actions 自动构建 Android、iOS 和 Windows 安装包。
- iOS 包会以 IPA 形式出现在 Release 附件中，适合 App Store Connect 或 TestFlight 流程。
