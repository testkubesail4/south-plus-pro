# v0.1.15

## 本版亮点

- 帖子正文现在支持更稳定的整段选中文本，复制长回复、整理资源说明和保存附件信息会顺手很多。
- 楼层里的顶部附件会回到正文下方展示，阅读顺序更接近原站，不会再出现“先看到附件、后看到正文”的割裂感。
- 图片加载失败时的占位和重试反馈更清楚，资源类帖子在网络不稳时更容易判断到底是链接失效还是加载失败。
- 修复部分出售帖里“购买框位置会在购买前后跳动”的问题，未购买和购买后现在都会保持和原站一致的内容顺序。

## 体验优化

- 这一版把帖子正文的文本、图片、附件、下载信息和出售区块整理到同一条渲染链里，复杂楼层的显示更稳定。
- 论坛图片缓存链路做了清理，列表页和详情页之间的图片复用更自然，重复加载和状态抖动更少。
- 新增和补强了帖子渲染、图片缓存、出售区块顺序相关测试，后续继续改阅读体验时更不容易把旧问题带回来。

## 修复

- 修复帖子正文可选中后，富文本、引用和图片混排时选区容易不完整的问题。
- 修复楼层顶部附件和正文分离过远的问题，现在会跟随正文一起出现在更合理的位置。
- 修复图片失败态提示不够明确的问题，失败占位、打开原图和复制链接等操作反馈更一致。
- 修复部分出售帖会把购买框错误挪到图片下方，购买成功刷新后又跳回图片上方的问题。
- 修复出售帖顺序修正后，购买提示内容可能被重复当作引用渲染的问题。
- 修复附件搬运和正文顺序渲染并存时，个别楼层会多出多余空行的问题。

## 技术说明

- 本版对帖子正文渲染做了一次较大的整理，核心目标是让正文顺序、附件位置、图片状态和出售区块都更接近论坛原始表现。
- 里面有一些渲染层和解析层的重构，但对外最直接的结果就是：阅读更顺、复制更稳、资源帖更不容易出现位置错乱。

## 发布说明

- 本版本继续由 GitHub Actions 自动构建 Android、iOS 和 Windows 安装包。
- Release 页面会继续提供 Android 多架构 APK、AAB、iOS sideload IPA、Windows x64 压缩包和 SHA256 校验文件。
