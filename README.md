## 这是什么

South Plus Pro 是一个非官方的南+论坛客户端，重点解决手机访问论坛时常见的不方便：页面拥挤、切换慢、回复麻烦、收藏和付费帖操作割裂。

它不会改变论坛本身的数据，也不会托管论坛内容。你仍然使用自己的南+账号登录，只是通过一个更适合移动端的界面完成浏览、回复、收藏、查看收藏和购买付费帖子。

## 适合谁

- 经常用手机浏览南+，觉得原网页在移动端不够顺手。
- 希望在手机上更方便地回复、引用、收藏和查看收藏。
- 会阅读或购买付费帖，希望操作完成后直接回到内容。
- 需要更稳定的访问线路，不想每次手动折腾网络入口。

## 基本使用

1. 下载并安装对应平台的版本。
2. 打开应用，在账号页登录自己的南+账号。
3. 从首页进入版块，或使用收藏页查看自己收藏过的主题。
4. 在帖子详情页直接回复、引用、收藏或购买付费内容。

## 说明

South Plus Pro 是第三方客户端，不是南+官方应用。项目只提供更适合移动端的访问体验，不提供、存储或分发论坛内容。请遵守所在地法律法规和南+论坛规则，使用自己的账号，并自行判断内容访问风险。
